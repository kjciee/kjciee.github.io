大会会場への交通案内

情報科学部がある枚方（ひらかた）キャンパスは，大阪，京都，奈良が隣接する地帯にあります。
 
電車･バスでのアクセス
  情報科学部の最寄りの駅はJR学研都市線長尾駅です．長尾駅から直通バスが運行されています．樟葉駅からも多数のバスが運行されています。

「大阪工大」または「北山中央」下車すぐ
所要時間：約30分

車でのアクセス
  車を運転される方は，１号線，またはそれに準じる道路をご利用下さい．京都からですと，宇治川，木津川を渡ったあと，洞ヶ峠というところにさしかかります．この峠を下りたところに「家具団地前」という交差点があります．ここで，左折します．３００メートルくらいで，初めての交差点があり，ここで右折です．坂をあがったところにキャンパスの入り口があります。
   
駐車に際してのお願い
  キャンパスには校舎の裏側に学生が利用する駐車用スペース (約６００台駐車可能)がありますので利用下さい．しかし，駐車場には駐車位置表示が無く，また，案内人が居ませんので，互いに他人の通行の妨げにならないよう配慮の上駐車下さい．また，校舎表側にも来客，職員用の駐車場がありますが，ここには来客，物資運搬のための車が出入りしますので駐車はご遠慮下さい。
